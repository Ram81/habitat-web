Instructions

To navigate through the room, you can use the following keyboard controls:

  • W: To move forward

You can look around using the following keyboard controls:

  • Left arrow/A: Turn left
  • Right arrow/D: Turn right
  • Up arrow: Look up
  • Down arrow: Look down

Instructions when submitting the task:

  • When you find an object, please move as close as possible to the object.
  • You can only submit the task when you are within 1m of the object.
